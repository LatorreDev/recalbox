KUBO 3
2020 SJ Games


Infos
-------

The game takes up the adventures of the cowboy turtle where the previous episode ended. Moglar, the evil mole kidnapped the villagers... and Kubo will have to find 4 sacred crystals in order to free them.

http://www.nesmakers.com/viewtopic.php?f=52&t=4817
Made with NESmaker: www.thenew8bitheroes.com


Use with FCEUX/MESEN or the FCEUMM core (retropie/recalbox)


Controls
--------

Left/Right	Move the player
Up/Down		Select player mode (on the Title Screen)

A 	Jump (on platformer screens)
B 	Shoot (when equiped)

Select 	Change the selection on the Title Screen
Start 	Start the game (on the Title Screen)
