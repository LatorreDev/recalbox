========================
!! Papi Commando in Ceplusplus Land !!
========================
* Sega Master System * Official Demo 

Don't wait to play it ! 

Play the 3 free Zone of Basic City in this free Plateformer Game's Demo !

Controls :

PAD : Move Papy ( You can use the DOWN direction to avoid ennemy or reach low passage. )

Button A : Jump.

Button B : Fire.

Button START : Pause Game / Life / Stuff Screen.
